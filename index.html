GIT

Git is a distributed version control system that is used to track changes to files and coordinate work on those files among multiple people.

MAVEN

Maven is a build automation tool used primarily for Java projects. Maven can also be used to build and manage projects written in C#, Ruby, Scala, and other languages.

JENKINS

Jenkins is an open-source automation server used to automate tasks related to building, testing, and deploying software.

DOCKER

Docker is a containerization platform that allows you to quickly build, test, and deploy applications as portable, self-sufficient containers that can run virtually anywhere.

ANSIBLE

Ansible is an open source automation platform used for IT tasks such as configuration management, application deployment, and orchestration. It can be used to manage both local and remote systems, and is designed to be very easy to use.
